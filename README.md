# Inet_Test
วิธีรันโปรเจกต์
ใช้ Live Server ใน VS Code
1. เปิดโฟลเดอร์โปรเจกต์ใน VS Code
2. ติดตั้ง extension ชื่อ Live Server
3. เปิดไฟล์ index.html
4. คลิกขวาแล้วเลือก Open with Live Server
5. เว็บจะเปิดขึ้นในเบราว์เซอร์อัตโนมัติ

โครงสร้างไฟล์หลัก
- index.html: หน้า Login
- index2.html: หน้า 1life systems
- style.css: สไตล์การแสดงผลหน้า Login
- style2.css: สไตล์การแสดงผลหน้า 1life systems
- scripts.js: JavaScript สำหรับ dropdown, tab, pagination, การจัดการสถานะ active และการสลับหน้า
- images/: ไฟล์รูปภาพที่ใช้ในหน้าเว็บ
